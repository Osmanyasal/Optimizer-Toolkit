\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classoptkit_1_1core_1_1BlockGroupProfiler}{Block\+Group\+Profiler}} class utilizes the RAII technique to initiate and conclude profiling for a specific raw event. Profiling commences upon instantiation and persists until the current scope is exited. 

Note that block profiler DOES group raw\+\_\+events! If you to monitor each event standalone please \mbox{\hyperlink{classoptkit_1_1core_1_1BlockProfiler}{Block\+Profiler}}

Each raw\+\_\+event creates seperate file\+\_\+description (fd) to read data and each raw\+\_\+event is treated seperately.~\newline
 In cases where CPU performs multiplexing and since each event treated seperately, you cannot gurantee that ~\newline
 events E1 and E2 will record the same instructions.~\newline
