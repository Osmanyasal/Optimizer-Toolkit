\doxysubsection{Detailed Description}
ASK CPU PMU and PMU related queries here~\newline
 This \mbox{\hyperlink{classoptkit_1_1core_1_1Query}{Query}} class uses {\bfseries{libpfm4}} to retrieve information. 

Don\textquotesingle{}t forget to call \mbox{\hyperlink{classoptkit_1_1core_1_1Query_ab4df06ae168f82a895eccb67e9d10436}{init()}} before using it and \mbox{\hyperlink{classoptkit_1_1core_1_1Query_a9851b438489ebf27e1ef0f7cba38e878}{destroy()}} when you\textquotesingle{}re done with it.~\newline
 These 2 method calls can be done at the beginning and end of the application.
